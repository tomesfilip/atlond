# Atlond

Atlond is a mobile app designed to connect people for sports activities. Whether you're planning a solo run, a cycling session, or a friendly football match, Atlond helps you find and connect with others who want to join you.

## MVP

In the MVP, there should be just a basic reflection of the main purpose of Atlond.

### Features

- **User Registration**: Users can sign up and log in to create and join activities.
- **Create Sports Activities**: Registered users can create activities specifying the date, time, location, type of sport, and duration.
- **View Activities**: Users can view a list of all sports activities. Logged-in users can see detailed information, while non-logged-in users can see basic info.
- **Join Activities**: Users can request to join activities, and creators can confirm participation.
- **Activity Details**: Detailed information is available for logged-in users.

### Activity Card Information

On the homepage, all current sports activities are displayed with basic information. Each activity card includes:

- **Title**: Brief, catchy title summarizing the activity.
- **Activity Type**: Type of sport (e.g., run, bike, football).
- **Date and Time**: When the activity is happening.
- **Location**: Where the activity is taking place.
- **See More Button**: Click to view more details (requires login).

Detailed view (visible upon clicking "See more" if logged in) includes:

- **Duration**: How long the activity will last.
- **Number of Participants**: Current number of participants / maximum number (if applicable).
- **Organizer's Name**: The name of the user who created the activity.
- **Join Button**: Allows users to request to join the activity.
